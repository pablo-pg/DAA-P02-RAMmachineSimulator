## Compilación y ejecución
Compilación:  
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```  
Después de compilar, puede ejecutar manualmente con ```$ ./simulador ram_program.ram input_tape.in output_tape.out```  
Además, tiene la opción `-h` y `--help`
